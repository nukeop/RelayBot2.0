Features

An advanced, modern Steam chatbot.

Extensible

Easily create your own self-contained plugins that react to events and are added to the bot automatically.

Free

Relay Bot 2.0 is being developed on Github under GNU GPL v3, which guarantees that it is, and will remain, free software.

Powerful

Relay Bot 2.0 is more than just a chatbot. Its features include accessing terminals, built-in user management, all directly from the Steam chat window. Advanced users can even evaluate python expressions for maximum control.
